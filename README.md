# 🧠 MatematIQ

**MatematIQ** adalah aplikasi latihan interaktif untuk siswa kelas 7 yang membantu memahami konsep matematika dasar seperti perkalian, pembagian, pecahan, dan waktu melalui kuis yang menyenangkan.

---

## ✨ Fitur

* 🎯 **Latihan Soal Interaktif**
  * Pilihan ganda & input jawaban
  * Sistem skor (benar, salah, streak)
  * Progress tracking per sesi
* 📖 **Materi & Rumus**
  * Penjelasan konsep matematika
  * Contoh soal & langkah penyelesaian
* ⚡ **Tantangan Cepat**
  * Mode 30 detik
  * Latih kecepatan berhitung
* 🧩 **Mode Variatif**
  * Perkalian & Pembagian
  * Pecahan & Desimal
  * Bilangan negatif
  * Membaca waktu (digital & analog)

## 📦 Instalasi (Local)

Clone repository:

```bash
git clone https://github.com/codeirawan/matematiq.git
cd matematiq
```

Buka file:

```bash

```
